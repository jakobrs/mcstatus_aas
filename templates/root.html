mcstatus wrapper server

List of endpoints

  • /server/<servername> - Same as /server/<servername>/status
  • /server/<servername>/status - Plaintext server status
  • /server/<servername>/json - JSON server status
  • /status?servername=...[&format=...] - generic endpoint where format is one of
    • status
    • plaintext - same as status
    • json
Made using Flask and mcstatus. https://github.com/jakobrs/mcstatus_aas
